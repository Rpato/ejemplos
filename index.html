Definiciones Clave en Desarrollo Web

Conceptos

HTML:

Lenguaje de marcado que estructura el contenido de una página web.

CSS:

Lenguaje de estilos que define la apariencia visual de los elementos HTML.

JavaScript:

Lenguaje de programación que añade interactividad y dinamismo a las páginas web.

Paleta de colores:

Conjunto de colores seleccionados para mantener coherencia estética en un diseño.

Frontend:

Parte de una aplicación web que interactúa directamente con el usuario.

Backend:

Parte de la aplicación que gestiona la lógica, datos y comunicación con el servidor.

Responsive Design:

Técnica de diseño que adapta la visualización de una página a diferentes dispositivos.
